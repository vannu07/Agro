## 📞 Quick Help

### "CSS not loading?"
→ Check layout.html for import statements
→ Verify file paths are correct
→ Clear browser cache

### "Animations not showing?"
→ Check animations.css is loaded after design-system.css
→ Verify no CSS conflicts
→ Check browser console for errors

### "Responsive not working?"
→ Check viewport meta tag
→ Test with actual devices, not just browser resize
→ Review breakpoints in design-system.css

### "Form validation not working?"
→ Check interactions.js is loaded
→ Verify form has required="required" attribute
→ Check browser console for JavaScript errors
