### 1. Start Flask App
```bash
cd app
python app.py
```

### 2. Open Browser

## 📞 Quick Help

**Can't see CSS?**
- Clear browser cache (Ctrl+Shift+Delete)
- Reload page (Ctrl+F5)

**Animations not working?**
- Try Chrome browser
- Check console for errors

**Want to customize?**
- Edit `/app/static/css/design-system.css` for colors
- Edit `/app/static/css/components.css` for components
